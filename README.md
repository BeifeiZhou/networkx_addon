networkx_addon
==============

### Some add-on modules to networkx library

1. Information propagation models
  (1) independent cascade model
  (2) linear threshold model

2. Vertex similarity measures
  (1) ASCOS (for both weighted and unweighted network)
  (2) Jaccard
  (3) Cosine
  (4) SimRank
  (5) RSS (r=2)
  (6) Katz
